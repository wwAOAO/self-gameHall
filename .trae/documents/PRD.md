## 1. 产品概述

一款经典扫雷游戏的网页版实现，使用 Vue 3 构建，提供流畅、美观、可自定义难度的扫雷体验。

- 核心功能：左键点击揭开格子，右键点击标记/取消标记旗帜，数字提示周围雷数，自动展开空白区域
- 目标用户：所有年龄段的休闲游戏玩家，怀旧经典游戏爱好者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 游客 | 无需注册 | 直接开始游戏，选择难度，查看成绩 |

### 2.2 功能模块

1. **游戏主界面**：雷区网格、计时器、剩余雷数显示、游戏状态提示
2. **难度选择**：初级（9×9，10雷）、中级（16×16，40雷）、高级（30×16，99雷）
3. **游戏控制**：重新开始、重置游戏
4. **交互反馈**：左键揭开、右键标旗、长按/双击快速展开（chord操作）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏页 | 雷区网格 | 根据难度生成不同大小的网格，每个格子有隐藏/数字/旗帜/地雷状态 |
| 游戏页 | 控制面板 | 显示剩余雷数、计时器、重新开始按钮、难度选择下拉 |
| 游戏页 | 游戏状态提示 | 胜利/失败时的弹窗或动画提示 |

## 3. 核心流程

用户打开页面 → 默认初级难度生成 → 点击格子开始游戏（计时启动） → 继续揭开/标旗 → 踩雷则游戏结束，全部非雷格子揭开则胜利 → 可随时切换难度或重新开始

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景 #1a1a2e 到 #16213e 渐变；卡片背景 #2a2a4a
- **强调色**：#e94560（红色，用于地雷和数字警示色）
- **数字颜色**：1-蓝色、2-绿色、3-红色、4-深蓝、5-深红、6-青蓝、7-黑色、8-灰色（经典扫雷配色）
- **字体**：中文使用系统 UI 字体，数字显示使用等宽字体
- **布局**：居中卡片设计，带有精致边框和阴影
- **动效**：点击时轻微缩放动画，掀开时的翻转效果，胜利/失败时的庆祝/爆炸粒子效果

### 4.2 页面设计总览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 游戏页 | 顶部标题 | 扫雷游戏名称，带图标装饰 |
| 游戏页 | 控制面板 | 剩余雷数（红色数字指示器）、笑脸重新开始按钮、计时器（三位数显示） |
| 游戏页 | 难度切换 | 三个按钮切换初级/中级/高级 |
| 游戏页 | 雷区网格 | 基于 CSS Grid 布局的单元格矩阵 |
| 游戏页 | 游戏结果 | 胜利/失败覆盖层，带半透明遮罩和文字提示 |

### 4.3 响应式

- PC 优先设计，雷区根据窗口大小自动调整格子尺寸
- 初级和中级难度在移动设备上也能良好显示
- 高级难度在移动设备上可横向滚动

## 5. 交互细节

- **首次点击必安全**：首次点击的格子及其周围 8 格都不会有地雷
- **旗帜模式**：右键点击切换旗帜标记，已标旗的格子不能左键揭开
- **Chord 操作**：点击已揭开的数字格子，如果周围旗帜数匹配则自动揭开剩余格子
- **游戏状态**：未开始、进行中、胜利、失败 四种状态
